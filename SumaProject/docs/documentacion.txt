La aplicación COBOL original se transforma en una aplicación Spring Boot moderna. El programa COBOL realiza una suma simple de dos números y muestra el resultado. La nueva aplicación Java utiliza Spring Boot para crear un API REST que también suma dos números enteros y devuelve el resultado.

Para lograr esto, se ha creado una clase SumaApplication que contiene el punto de entrada principal de la aplicación y configura el entorno Spring Boot. Dentro de esta clase, se ha inyectado un controlador REST llamado SumaController, el cual expone un endpoint `/suma` que acepta dos parámetros numéricos y devuelve su suma.

Las pruebas unitarias se han diseñado para verificar el comportamiento del controlador, asegurando que la operación de suma funcione correctamente.

Esta transformación permite una aplicación más escalable y fácil de mantener, con las ventajas de la programación orientada a objetos y los beneficios de un framework moderno como Spring Boot. La lógica de negocio se ha separado claramente de la capa de presentación, permitiendo futuras expansiones y reconfiguraciones.
