El siguiente código Java es una traducción moderna del código COBOL proporcionado. Utiliza Spring Boot para la configuración básica y define una clase `SumaApplication` que contiene el método `suma` para realizar la adición de dos números. También se proporciona una clase de prueba unitaria `SumaTest` que verifica el método de suma con un caso de prueba simple.

El código utiliza las últimas prácticas y tecnologías de Java, incluyendo anotaciones de Spring Boot para la configuración y JUnit para las pruebas unitarias. Esto garantiza que el código sea fácil de mantener y ampliar en el futuro. La estructura clara y modular permite una buena legibilidad y comprensión.
