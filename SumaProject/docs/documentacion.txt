Suma de dos números

La aplicación suma dos números enteros y devuelve el resultado.

Para ejecutar la aplicación:
1. Asegúrate de tener Java 17 o superior y Spring Boot 3 instalado.
2. Ejecuta el comando `java -jar your-jar-name.jar`

Para desplegar las API en local:
1. Ejecuta `spring-boot:run` en un IDE compatible con Spring Boot.
2. Accede a las API en http://localhost:8080

Las API expuestas son:
- `GET /suma?num1={num1}&num2={num2}` : Suma dos números y devuelve el resultado.

Pruebas:
- `TestContextManager` : Inicializa el contexto de Spring Boot.
- `SumaControllerTest` : Prueba unitaria que verifica el cálculo de suma.
