La aplicación COBOL original se transforma en una aplicación Spring Boot que proporciona un API REST para sumar dos números. El código Java utiliza Spring Boot para permitir una configuración mínima y se proporcionan pruebas unitarias para validar la funcionalidad del controlador. El diagrama de clases Mermaid muestra la relación entre las clases principales.

La aplicación Spring Boot se inicia mediante la clase SumaApplication, que está anotada con @SpringBootApplication. La clase SumaController es un controlador REST que maneja las solicitudes GET a la ruta "/suma" y devuelve la suma de dos números. Se ha incluido una prueba unitaria para validar el comportamiento del método suma.
