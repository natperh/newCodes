# CRUD en COBOL

Este código COBOL implementa un menú simple de CRUD (Create, Read, Update, Delete) para gestionar una base de datos de usuarios.

## Reglas de Negocio
- Un usuario se compone de un ID numérico de 4 dígitos y un nombre de texto.
- Los datos de usuario se almacenan en un archivo llamado 'usuarios.dat' en formato secuencial de líneas.
- Las opciones del menú son: 1-Crear, 2-Leer, 3-Actualizar, 4-Borrar, 5-Salir.
- La opción de creación pide al usuario un ID y un nombre, y los escribe en el archivo.
- La opción de lectura muestra todos los registros de usuario uno por uno hasta que el usuario decide salir.

## Propósito
Este programa demuestra el uso básico de COBOL para manejar archivos y realizar operaciones CRUD sencillas.
