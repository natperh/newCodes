Este código COBOL implementa un menú simple de CRUD (Create, Read, Update, Delete) para gestionar registros de usuarios en un archivo llamado "usuarios.dat". Cada registro contiene un ID de 4 dígitos y un nombre de 20 caracteres.

La sección DATA define la estructura de los registros de usuario, mientras que la sección WORKING-STORAGE contiene variables para controlar el flujo del programa.

La sección PROCEDURE contiene la lógica del programa, que muestra un menú al usuario y realiza operaciones basadas en la selección del usuario: crear un nuevo registro, leer todos los registros, actualizar un registro existente o salir del programa.
